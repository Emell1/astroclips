# AstroClips Design System

## Vibe
Dark, professional, cinematic. Herramienta de producción de contenido. No es una app de consumo — es una workstation.

## Colors
- Background: `#0a0a0f` (casi negro con tinte azul)
- Surface: `#12121a`
- Surface elevated: `#1a1a26`
- Border: `#2a2a3a`
- Primary: `#7c3aed` (violeta)
- Primary hover: `#6d28d9`
- Accent: `#f59e0b` (ámbar — para scores y highlights)
- Success: `#10b981`
- Danger: `#ef4444`
- Text primary: `#f0f0f5`
- Text secondary: `#8888a0`
- Text muted: `#55556a`

## Typography
- Display: `Syne` (Google Fonts) — para títulos grandes
- Body: `DM Sans` — legible, moderno, no genérico
- Mono: `JetBrains Mono` — para timestamps, códigos

## Layout
- Max width: 1280px
- Sidebar izquierda para navegación (si aplica)
- Cards con borde sutil, sin sombras exageradas
- Spacing generoso

## Components
- Botones: rounded-lg, fondo primary, sin outline excesivo
- Cards: bg surface, border, rounded-xl
- Progress bars: violeta con glow sutil
- Badges de score: ámbar para alto, verde para medio
- Upload zone: dashed border, drag & drop visual feedback
